# Original Rodney

These files recreate the original files from the book, Build Your Own
Self-Programming Robot by David L Heiserman.  The original, 43 years old
at the time of writing this, is a _bit_ dated.  So there is no plan to
actually build anything from the original files.  But, since he doesn't
present an entire schematic anywhere in the book, I wanted to put that
together and gather it all in once place.  There are three reasons for that.

1.  Posterity.  Just nice to have it in case someone wants it.
2.  Reference.  Makes it easier to understand the discussion in the book.
